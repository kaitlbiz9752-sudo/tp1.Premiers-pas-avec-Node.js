# TP Node.js – Création d’une API complète avec Express.js

## Description du projet
Ce TP vous guide pas à pas dans la création d’une application Node.js complète comprenant :

- Un serveur HTTP basique
- Une API REST capable de gérer une liste d’utilisateurs
- La manipulation de fichiers côté serveur
- L’utilisation du framework Express.js
- La gestion des variables d’environnement
- L’ajout d’un middleware de journalisation
- Une interface utilisateur simple en HTML/CSS/JS

L’objectif est de comprendre les modules natifs de Node.js ainsi que les bases du développement backend moderne.

## Étape 1 — Installation de Node.js
- Télécharger Node.js depuis nodejs.org
- Installer la version LTS
- Vérifier l’installation via :
  - node -v
  - npm -v

## Étape 2 — Création du projet
- Création d’un dossier dédié
- Initialisation d’un projet Node.js avec npm
- Génération du fichier package.json

## Étape 3 — Création d’un serveur HTTP simple
- Utilisation du module natif http
- Mise en place d’un serveur local accessible sur http://localhost:3000

## Étape 4 — Manipulation de fichiers
- Création d’un fichier texte
- Lecture asynchrone et affichage de son contenu via le serveur

## Étape 5 — Création d’une API REST
- Route pour obtenir tous les utilisateurs
- Route dynamique pour obtenir un utilisateur par ID
- Gestion des erreurs (404, erreurs serveur)
- Analyse des URLs

## Étape 6 — Passage à Express.js
- Installation d’Express.js
- Création d’une API structurée :
  - /api/users : obtenir tous les utilisateurs
  - /api/users/:id : obtenir un utilisateur par ID
  - POST /api/users : ajouter un utilisateur
- Middleware pour traiter les requêtes JSON
